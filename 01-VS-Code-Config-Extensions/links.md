## Configs:

- Enable AutoSave (File -> Auto Save)
- Enable Format On Save (Settings -> User -> Text Editor -> Formatting)
- Format On Save Mode -> File
- Color Theme: One Dark Pro
- File Icon Theme: Material Icon Theme
